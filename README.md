VitaMatch is a full-stack web application designed to streamline and digitize the organ donation and allocation process between **donors**, **doctors**, and **hospitals**.

The platform enables donors to register and offer organs, doctors to request and allocate organs, and both parties to track donation status in real time.

---

## Features

### Authentication
- JWT based login/signup
- Role based access (Donor / Doctor)

---

### Donor Dashboard
- View available organ requests
- Donate organs voluntarily or to hospital requests
- Provide consent (Living / Post-death)
- Track donation status
- Confirm or reject allocations

---

### Doctor Dashboard
- View hospital requests
- Request organs
- View available donors
- Accept organs
- Track allocations
- Dashboard analytics (My Requests + Hospital Requests)

---

### Security
- JWT authentication
- Protected routes
- Role validation
- Secure API headers
